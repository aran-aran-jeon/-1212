# 🌐 구구단 게임 공유하기 가이드

친구들에게 게임을 공유하는 가장 쉬운 방법들입니다!

## 방법 1: Netlify Drop (가장 쉬움! ⭐ 추천)

**장점**: 계정 없이도 가능, 1분이면 끝!

### 단계:
1. 브라우저에서 https://app.netlify.com/drop 접속
2. `구구단 게임 만들기` 폴더 전체를 드래그 앤 드롭
3. 잠시 기다리면 자동으로 URL이 생성됨!
4. 그 URL을 친구들에게 공유하면 끝!

**예시 URL**: `https://random-name-123.netlify.app` 같은 형식으로 생성됩니다.

---

## 방법 2: Vercel (또는 간단함)

**장점**: 빠르고 안정적

### 단계:
1. https://vercel.com 접속 (GitHub 계정으로 간단 가입)
2. "Add New Project" 클릭
3. `구구단 게임 만들기` 폴더를 드래그 앤 드롭
4. 자동으로 배포되고 URL 생성!
5. URL 공유하기

---

## 방법 3: GitHub Pages (무료, 영구적)

**장점**: 완전 무료, 영구적으로 사용 가능

### 단계:
1. https://github.com 접속해서 계정 만들기
2. 새 저장소(Repository) 만들기
   - 이름: `gugudan-game` (아무거나 상관없음)
   - Public으로 설정
3. `구구단 게임 만들기` 폴더 안의 파일들을 업로드
   - `index.html`, `style.css`, `game.js` 모두 업로드
4. 저장소 Settings → Pages 메뉴로 이동
5. Source를 "main" 브랜치로 선택
6. 저장하면 몇 분 후 URL 생성!
   - 형식: `https://[당신의아이디].github.io/gugudan-game/`

---

## 방법 4: Surge.sh (터미널 사용 가능하면)

**장점**: 매우 빠름, 커스텀 도메인 가능

### 단계:
1. 터미널/명령 프롬프트 열기
2. `npm install -g surge` 실행
3. `구구단 게임 만들기` 폴더로 이동
4. `surge` 실행
5. 이메일 입력, 비밀번호 설정
6. 자동으로 URL 생성!

---

## 📱 공유 팁

배포 후 받은 URL을:
- 카카오톡으로 공유
- 인스타그램 스토리에 링크 공유
- 블로그에 임베드
- QR 코드로 만들어서 공유

---
