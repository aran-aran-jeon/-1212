# 📘 구구단 몬스터 배틀 게임 기획서

## 1. 게임 개요

* **장르**: 교육형 미니 RPG / 구구단 퀴즈 배틀
* **핵심 컨셉**: 구구단 문제를 맞혀서 몬스터를 공격하고, 틀리면 몬스터가 플레이어를 공격한다.
* **승리 조건**: 몬스터 체력(HP)을 0으로 만들면 승리
* **패배 조건**: 플레이어 체력(HP)을 0으로 만들면 패배

---

## 2. 기본 규칙

### 2.1 체력 시스템

* 플레이어 HP: **100**
* 몬스터 HP: **100**
* 한 번 공격 시 피해량: **10**
* 공격은 “정답/오답”에 따라 자동 발생

### 2.2 턴 흐름

1. 문제 출제 (구구단)
2. 플레이어가 답 입력
3. 정답이면 → 몬스터 HP 감소
4. 오답이면 → 플레이어 HP 감소
5. HP 확인 후 승/패 처리
6. 다음 문제로 반복

---

## 3. 문제(구구단) 시스템

### 3.1 출제 방식

* 2~9단 랜덤
* 곱하는 수 역시 1~9 랜덤
* 예: `7 × 3 = ?`

### 3.2 난이도 옵션 (선택 기능)

* **Easy**: 2~5단만
* **Normal**: 2~9단 전체
* **Hard**: 제한 시간 추가 + 연속성 강화

---

## 4. 콤보 시스템

### 4.1 콤보 조건

* 연속 정답 시 콤보 증가
* 오답 시 콤보 0으로 초기화

### 4.2 콤보 효과

* 콤보가 쌓이면 **공격 데미지가 증가**
* 화면에 **콤보 표시(UI)** 등장

### 4.3 데미지 증가 공식(안)

* 기본 데미지: `10`
* 콤보 데미지 보너스: `콤보 수 × 2`
* 최종 데미지:

  ```
  damage = 10 + combo * 2
  ```
* 예시

  * 0콤보 → 10 피해
  * 3콤보 → 16 피해
  * 5콤보 → 20 피해

> (Cursor로 구현할 때 깔끔하고 직관적인 수식이라 추천)

### 4.4 콤보 UI

* 정답 시 화면 중앙 혹은 몬스터 위에 표시:

  * `COMBO x2`
  * `COMBO x5!` (강조 효과)

## 5. UI/UX 구성

### 5.1 필수 UI 요소

1. **플레이어 체력바(HP Bar)**

   * 좌측 하단 혹은 좌측 상단
2. **몬스터 체력바(HP Bar)**

   * 우측 상단
3. **문제 표시 영역**

   * 중앙 상단
   * 예: `8 × 4 = ?`
4. **입력 창**

   * 중앙
5. **콤보 표시**
6. **결과 로그(텍스트)**

   * 예:

     * “정답! 몬스터에게 16 데미지!”
     * “오답… 플레이어가 10 데미지를 받았다.”

### 5.2 화면 흐름

* **Start 화면 → 게임 화면 → 결과 화면(승리/패배)**

---

## 6. 게임 상태(State) 정의

### 6.1 주요 상태

* `start`
* `playing`
* `win`
* `lose`

### 6.2 상태 전환

* start → playing : 게임 시작 버튼 클릭
* playing → win : 몬스터 HP <= 0
* playing → lose : 플레이어 HP <= 0
* win/lose → start : 다시하기 버튼

## 7. 데이터 구조 설계(예시)

```js
player = {
  hp: 100,
  combo: 0
}

monster = {
  hp: 100
}

problem = {
  a: 2~9 랜덤,
  b: 1~9 랜덤,
  answer: a*b
}
```

---

## 8. 주요 함수/로직 정의

### 8.1 문제 생성

```js
function generateProblem() {
  const a = random(2,9)
  const b = random(1,9)
  return { a, b, answer: a*b }
}
```

### 8.2 정답 판정

```js
function checkAnswer(input, problem) {
  return parseInt(input) === problem.answer
}
```

### 8.3 공격 처리

```js
function attack(target, damage) {
  target.hp = Math.max(0, target.hp - damage)
}
```

### 8.4 콤보 기반 데미지 계산

```js
function calcDamage(combo) {
  return 10 + combo * 2
}
```

### 8.5 턴 처리

```js
if (isCorrect) {
  player.combo += 1
  let dmg = calcDamage(player.combo)
  attack(monster, dmg)
} else {
  player.combo = 0
  attack(player, 10)
}
```
