# 🎮 구구단 몬스터 배틀 게임

구구단 문제를 맞혀서 몬스터를 물리치는 교육형 미니 RPG 게임입니다!

## 🎯 게임 방법

1. 구구단 문제가 출제됩니다
2. 정답을 입력하면 몬스터에게 데미지!
3. 오답을 입력하면 플레이어가 데미지를 받습니다
4. 연속 정답 시 콤보가 쌓여 더 큰 데미지를 줄 수 있습니다
5. 몬스터의 HP를 0으로 만들면 승리!

## 🚀 실행 방법

### 로컬에서 실행
1. `index.html` 파일을 더블클릭하거나
2. 브라우저로 드래그 앤 드롭

### 웹에 배포하기
`배포 가이드.md` 파일을 참고하세요!

## 📁 파일 구조

```
구구단 게임 만들기/
├── index.html      # 게임 HTML 구조
├── style.css       # 스타일 및 애니메이션
├── game.js         # 게임 로직
└── monster.png     # 몬스터 이미지 (선택사항)
```

## ✨ 주요 기능

- ✅ 랜덤 구구단 문제 출제 (2~9단)
- ✅ HP 시스템 (플레이어/몬스터 각 100HP)
- ✅ 콤보 시스템 (연속 정답 시 데미지 증가)
- ✅ 공격 애니메이션
- ✅ 승리/패배 처리

## 🎨 특징

- 반응형 디자인 (모바일/PC 모두 지원)
- 부드러운 애니메이션
- 직관적인 UI/UX
